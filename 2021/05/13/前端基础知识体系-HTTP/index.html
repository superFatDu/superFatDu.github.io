HTTPеҚҸи®®

HTTPжҳҜдёҖз§ҚдёҚдҝқеӯҳзҠ¶жҖҒпјҢеҚіж— зҠ¶жҖҒпјҲstatelessпјүеҚҸи®®гҖӮеҚҸи®®еҜ№дәҺеҸ‘йҖҒиҝҮзҡ„иҜ·жұӮжҲ–иҖ…е“Қеә”йғҪдёҚеҒҡжҢҒд№…еҢ–еӨ„зҗҶгҖӮ

HTTPж–№жі•

  1. GETпјҡиҺ·еҸ–иө„жәҗ
  2. POSTпјҡдј иҫ“е®һдҪ“дё»дҪ“
  3. PUTпјҡдј иҫ“ж–Үд»¶пјҲеғҸFTPдёҠдј ж–Үд»¶пјӣ204 No Contentпјү
  4. HEADпјҡиҺ·еҸ–жҠҘж–ҮйҰ–йғЁпјҲзЎ®и®ӨURIзҡ„жңүж•ҲжҖ§пјҢиө„жәҗжӣҙж–°ж—¶й—ҙзӯүпјү
  5. DELETEпјҡеҲ йҷӨж–Үд»¶пјҲ204 No Contentпјү
  6. OPTIONSпјҡиҜўй—®ж”ҜжҢҒзҡ„ж–№жі•
  7. TRACEпјҡиҝҪиёӘи·Ҝеҫ„
  8. CONNECTпјҡиҰҒжұӮз”Ёйҡ§йҒ“еҚҸи®®иҝһжҺҘд»ЈзҗҶпјҲиҰҒжұӮеңЁд»ЈзҗҶжңҚеҠЎеҷЁиҝӣиЎҢйҖҡдҝЎж—¶е»әз«Ӣйҡ§йҒ“пјҢе®һзҺ°з”Ёйҡ§йҒ“еҚҸи®®иҝӣиЎҢTCPйҖҡдҝЎпјҲSSLгҖҒTLSпјүпјү

Chunked Transfer Coding

HTTPйҰ–йғЁ

HTTPеҚҸи®®зҡ„иҜ·жұӮе’Ңе“Қеә”жҠҘж–Үдёӯеҝ…е®ҡеҢ…еҗ«HTTPйҰ–йғЁгҖӮйҰ–йғЁеҶ…е®№дёәе®ўжҲ·з«Ҝе’ҢжңҚеҠЎеҷЁеҲҶеҲ«еӨ„зҗҶиҜ·жұӮе’Ңе“Қеә”жҸҗдҫӣжүҖйңҖзҡ„дҝЎжҒҜгҖӮ

йҖҡз”ЁйҰ–йғЁеӯ—ж®ө

иҜ·жұӮйҰ–йғЁеӯ—ж®ө

е“Қеә”йҰ–йғЁеӯ—ж®ө

е®һдҪ“йҰ–йғЁеӯ—ж®ө

HTTPзҠ¶жҖҒз Ғ

1xxпјҲдёҙж—¶е“Қеә”пјү

  1. 100пјҡиҜ·жұӮиҖ…еә”еҪ“з»§з»ӯжҸҗеҮәиҜ·жұӮ
  2. 101пјҡеҲҮжҚўеҚҸи®®гҖӮиҜ·жұӮиҖ…е·ІиҰҒжұӮжңҚеҠЎеҷЁеҲҮжҚўеҚҸи®®пјҢжңҚеҠЎеҷЁе·ІзЎ®и®Өе№¶еҮҶеӨҮеҲҮжҚўгҖӮ

2xxпјҲжҲҗеҠҹпјү

  1. 200пјҡиҜ·жұӮжҲҗеҠҹе№¶иҝ”еӣһжӯЈзЎ®жҺҘеҸЈз»“жһң
  2. 201пјҡиҜ·жұӮжҲҗеҠҹе№¶жҲҗеҠҹеҲӣе»әдәҶиө„жәҗ
  3. 202пјҡжңҚеҠЎеҷЁе·ІжҺҘж”¶иҜ·жұӮпјҢдҪҶе°ҡжңӘеӨ„зҗҶгҖӮпјҲеҸҜиҪ®и®ӯз»§з»ӯиҜ·жұӮпјү
  4. 203пјҡйқһжҺҲжқғдҝЎжҒҜпјҢжңҚеҠЎеҷЁе·ІжҲҗеҠҹеӨ„зҗҶиҜ·жұӮпјҢдҪҶиҝ”еӣһзҡ„з»“жһңеҸҜиғҪжқҘиҮӘеҸҰдёҖдёӘжәҗ
  5. 204пјҡж— еҶ…е®№пјҢжңҚеҠЎеҷЁе·ІжҲҗеҠҹеӨ„зҗҶиҜ·жұӮпјҢдҪҶжІЎжңүиҝ”еӣһд»»дҪ•еҶ…е®№пјҲPUTпјү
  6. 205пјҡжҲҗеҠҹеӨ„зҗҶдәҶиҜ·жұӮпјҢж— еҶ…е®№пјҢдё”иҰҒжұӮиҜ·жұӮиҖ…йҮҚзҪ®ж–ҮжЎЈи§Ҷеӣҫ
  7. 206пјҡжңҚеҠЎеҷЁе·Із»ҸеӨ„зҗҶдәҶйғЁеҲҶGETиҜ·жұӮгҖӮпјҲRangeпјү

3xxпјҲйҮҚе®ҡеҗ‘пјү

  1. 300пјҡMultiple ChoicesпјҢиҜ·жұӮжҲҗеҠҹпјҢиҜ·жұӮзҡ„иө„жәҗеҸҜеҢ…жӢ¬еӨҡдёӘдҪҚзҪ®
  2. 301пјҡиҜ·жұӮжҲҗеҠҹпјҢдҪҶиө„жәҗиў«ж°ёд№…иҪ¬з§»еҲ°ж–°дҪҚзҪ®
  3. 302пјҡдёҙж—¶иҪ¬з§»пјҢиҜ·жұӮзҡ„зҪ‘йЎөдёҙж—¶и·іиҪ¬еҲ°е…¶д»–йЎөйқўпјҢдёҙж—¶йҮҚе®ҡеҗ‘
  4. 303пјҡеҜ№еә”еҪ“еүҚиҜ·жұӮзҡ„е“Қеә”еҸҜд»ҘеңЁеҸҰдёҖдёӘ URI дёҠиў«жүҫеҲ°пјҢдё”дҪҝз”ЁGETи®ҝй—®ж–°зҡ„ең°еқҖжқҘиҺ·еҸ–иө„жәҗ
  5. 304пјҡиҜ·жұӮзҡ„иө„жәҗжІЎжңүиў«дҝ®ж”№иҝҮпјҲзј“еӯҳпјү
  6. 305пјҡдҪҝз”Ёд»ЈзҗҶпјҢиҜ·жұӮиҖ…еә”иҜҘдҪҝз”Ёд»ЈзҗҶи®ҝй—®иҜҘйЎөйқў
  7. 306пјҡдёҙж—¶иҪ¬з§»пјҢеңЁжңҖж–°зҡ„и§„иҢғдёӯпјҢе·Із»ҸдёҚдҪҝз”Ё

4xxпјҲиҜ·жұӮй”ҷиҜҜпјү

  1. 400пјҡиҜ·жұӮеҮәзҺ°й”ҷиҜҜпјҲиҜӯд№үжңүиҜҜжҲ–иҖ…иҜ·жұӮеҸӮж•°жңүиҜҜпјү
  2. 401пјҡжІЎжңүжҸҗдҫӣи®ӨиҜҒдҝЎжҒҜпјҲAuthorizationпјү
  3. 402пјҡйў„з•ҷзҠ¶жҖҒз Ғ
  4. 403пјҡжңҚеҠЎеҷЁзҗҶи§ЈиҜ·жұӮпјҢдҪҶжӢ’з»қжү§иЎҢ
  5. 404пјҡиҜ·жұӮзҡ„иө„жәҗжңӘеңЁжңҚеҠЎеҷЁдёҠеҸ‘зҺ°
  6. 405пјҡж–№жі•зҰҒз”ЁпјҢжңҚеҠЎеҷЁзҰҒз”ЁдәҶиҜ·жұӮжҢҮе®ҡзҡ„ж–№жі•пјҲOptionпјү
  7. 406пјҡиҜ·жұӮзҡ„иө„жәҗзҡ„еҶ…е®№зү№жҖ§ж— жі•ж»Ўи¶іиҜ·жұӮеӨҙдёӯзҡ„жқЎд»¶пјҢеӣ иҖҢж— жі•з”ҹжҲҗе“Қеә”е®һдҪ“
  8. 407пјҡиҜ·жұӮиҖ…йңҖиҰҒд»ЈзҗҶжҺҲжқғпјҲProxy-Authenticateпјү
  9. 408пјҡжңҚеҠЎеҷЁиҜ·жұӮи¶…ж—¶
    вҖҰ

5xxпјҲжңҚеҠЎеҷЁй”ҷиҜҜпјү

  1. 500пјҡжңҚеҠЎеҷЁеҶ…йғЁй”ҷиҜҜпјҢж— жі•е®ҢжҲҗиҜ·жұӮ
  2. 501пјҡиҜ·жұӮиҝҳжІЎжңүиў«е®һзҺ°пјҢжңҚеҠЎеҷЁдёҚе…·еӨҮе®Ңж•ҙзҡ„иҜ·жұӮеҠҹиғҪ
  3. 502пјҡй”ҷиҜҜзҪ‘е…іпјҢжңҚеҠЎеҷЁдҪңдёәзҪ‘е…іжҲ–иҖ…д»ЈзҗҶпјҢд»ҺдёҠжёёжңҚеҠЎеҷЁж”¶еҲ°ж— ж•Ҳе“Қеә”
  4. 503пјҡжңҚеҠЎдёҚеҸҜз”ЁпјҲи¶…иҪҪжҲ–з»ҙжҠӨпјү
  5. 504пјҡзҪ‘е…іи¶…ж—¶пјҢжңҚеҠЎеҷЁдҪңдёәзҪ‘е…іжҲ–иҖ…д»ЈзҗҶпјҢжІЎжңүеҸҠж—¶д»ҺдёҠжёёж”¶еҲ°иҜ·жұӮ
  6. 505пјҡHTTPеҚҸи®®зүҲжң¬дёҚж”ҜжҢҒ

е…¶д»–

жҹҘзңӢе®ҢжҲҗдёҖдәӣзҡ„иҜ·жҢӘжӯҘпјҡHTTPзҠ¶жҖҒз Ғ

HTTPS

HTTPSиў«з§°дёәпјҡHTTP SecureпјҲи¶…ж–Үжң¬дј иҫ“е®үе…ЁеҚҸи®®пјүжҲ–HTTP Over SSLгҖӮ

HTTPзҡ„зјәзӮ№

  1. йҖҡдҝЎдҪҝз”ЁжҳҺж–ҮпјҲдёҚеҠ еҜҶпјүпјҢеҶ…е®№е®№жҳ“иў«зӘғеҗ¬ =гҖӢSSL/TLS
  2. дёҚйӘҢиҜҒ йҖҡдҝЎж–№зҡ„иә«д»ҪпјҢеҸҜиғҪйҒӯйҒҮдјӘиЈ… =гҖӢиҜҒд№Ұ
  3. ж— жі•йӘҢиҜҒ жҠҘж–Үе®Ңж•ҙжҖ§пјҢжңүеҸҜиғҪе·Із»Ҹиў«зҜЎж”№ =гҖӢPGP/MD5/SHA-1

йҖҡдҝЎеҠ еҜҶ

  1. SSL(Secure Socket LayerпјҢе®үе…ЁеҘ—жҺҘеұӮ)
  2. TLS(Transport Layer SecurityпјҢе®үе…Ёдј иҫ“еұӮеҚҸи®®)

жҖ»з»“

HTTP + еҠ еҜҶпјҲSSL/TLSпјү + и®ӨиҜҒпјҲиҜҒд№Ұпјү+ е®Ңж•ҙжҖ§дҝқжҠӨпјҲPGP/MD5пјү= HTTPS
